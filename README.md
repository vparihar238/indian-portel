# Indian Portal Website

This is the website for **Indian Portal — Aapka Digital India Partner**.

## Local run
1. npm install
2. npm run dev
3. Open http://localhost:3000

## Environment
- NEXT_PUBLIC_APK_LINK — public link to APK (hosted on Drive / S3 / storage)
- NEXT_PUBLIC_API_URL — optional external API endpoint (defaults to internal /api/forms)

## Deploy
1. Push to GitHub
2. On Vercel: Import Project from GitHub and Deploy
3. Set Environment Variables on Vercel (if needed)
